# 玄玄一键宏
魔兽世界脚本
- 基于robotgo设计的脚本，原理是在技能序列循环里，把每一个技能转变成按键，就是模拟手动按键。
- 暂时不能根据BUFF施放技能，只能运行固定序列。

### 编译
- 使用如下命令，可生成不带DOS黑窗口的exe文件
```
go build -ldflags -H=windowsgui
```
